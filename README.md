# BlinkerLED
Simple board with few blinking LEDs. The board creates a LED blinking effect, making it perfect for beginners who want to learn the basics of electronics, soldering, and PCB assembly. It is easy to make and can be used as cool decoration for your room.

# PCB & Schematic
